Helpful tools I came across while creating this Portofolio

CSS Templates

I found that looking at and modifying existing CSS templates can teach you a great deal on styling and positioning elements of a website. As a basis for my website, I started from this template and modified it to my needs.

I removed the elements I did not need and adjusted others so they would fit my project. Positioning of elements and the CSS box model can be very tricky at times.

HTML iFrames

I used iframes to structure and separate my content. While navigating via iframes comes with the disadvantage of disabling the browsers own navigation (no more going back a page), they let me to separate layout from content so I could keep a got overview on everything and allowed me to apply a lot of interactivity with ease.

The fact that the use of iframes is debatable and comes with controversial opinions can be seen here and here.

Method Draw

I used Method Draw to create the two icons in the navigation bar which enable changing the size of and swapping the assignment and code windows. You can create your image on a drawing surface and then export the result as an SVG file. It is also possible to import images and convert them to SVG.

SVGs are extremely light-weight and provide great flexibility and interactivity since all elements of the image can be accessed individually. Hovering over my buttons for instance only causes the inner background of the image to turn orange.

Adobe Kuler

Adobe Kuler proved to be an excellent tool to create and find good colour themes. You can either browse for colour themes created by other users by keyword or use the colour wheel you create your own.

Choosing the right colours for my page was not an easy task at all and I learnt that even the slightest change in colour can make a huge difference in the end. Colour coding is a science in its own right...

Google Prettify

I formatted the code displayed on my pages with the GooglePrettify plug-in, using the 'Tomorrow Night' template created by Chris Kempson.

The tool knows a lot of programming languages, analyses them and does appropriate syntax highlighting. Very helpful indeed as otherwise my code could never have looked that pretty.

ToolTipster

ToolTipster is an easy-to-use jQuery plug-in to create and style tooltips. I found it easy to set up and it comes with a good tutorial.

Although I did style my tooltips, I stayed far below the possibilities it has to offer. It gives you the option to define animations, gives you great control over position and even lets you create interactive tooltips.
