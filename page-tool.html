Helpful tools I came across while creating this Portofolio

CSS Templates

I found that looking at and modifying existing CSS templates can teach you a great deal on styling and positioning elements of a website. As a basis for my website, I started from this template and modified it to my needs.

I removed the elements I did not need and adjusted others so they would fit my project. Positioning of elements and the CSS box model can be very tricky at times.

HTML iFrames

I used iframes to structure and separate my content. While navigating via iframes comes with the disadvantage of disabling the browsers own navigation (no more going back a page), they let me to separate layout from content so I could keep a got overview on everything and allowed me to apply a lot of interactivity with ease.

The fact that the use of iframes is debatable and comes with controversial opinions can be seen here and here.
