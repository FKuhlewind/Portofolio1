Second Assignment: Handlebars

Create a 'Student' collection in JavaScript using appropriate data types. Then, use jQuery and Handlebars to display elements of this collection.

Extension:

I have stored the data in Firebase and added the functionality to add and remove students and to modify student information.

Thoughts on Handlebars...

I must admit that at first I could not see the true advantages of Handlebars. 'Why not simply use iteration to achieve the same?' was my initial thought. Through using it I discovered how its templating ability could be extended past the creation of tables to entire website structure and layout defined by and filled with changing data and content.
